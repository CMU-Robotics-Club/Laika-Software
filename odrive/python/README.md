# Odrive Control using python
basic control and testing using python and can.

Version used: 0.6.11-1

## Dependencies
Install this stuff
### Virtual Environment
On ubuntu, some Dependencies cannot be intalled using apt, so you need to use pip in a virtual environment. Run the following commands in this folder:
```sh
python -m venv .venv
source .venv/bin/activate
```
### pip:
```sh
pip install python-can odrive odrivetool
```

Using apt:

```sh
sudo apt install python3-can
```
